## Описание проекта

Проект направлен на создание интеллектуального помощника (ИИ-ассистента) для компании, занимающейся установкой окон. Цель ИИ-ассистента — автоматизировать взаимодействие с клиентами, обеспечить быстрые и точные ответы на вопросы, а также упростить процесс записи на встречи.

Благодаря использованию современных технологий искусственного интеллекта, ИИ-ассистент способен вести осмысленные диалоги, отвечать на часто задаваемые вопросы из базы знаний и выполнять практические задачи, такие как запись клиентов на встречу и добавление событий в Google Календарь.

---

## Функционал ИИ-ассистента

1. **Осмысленное ведение диалога**  
   - Ассистент использует нейросетевые технологии для поддержания естественного общения с клиентами.  
   - Способен понимать контекст запросов и адаптировать ответы под потребности пользователя.  

2. **Ответы на вопросы по базе знаний**  
   - Встроена база знаний, содержащая информацию о продуктах и услугах компании (например, типы окон, цены, сроки установки).  
   - Ответы формируются на основе актуальных данных, обеспечивая высокую точность информации.  

3. **Запись клиента на встречу**  
   - ИИ-ассистент может предложить клиенту удобное время для встречи.  
   - После согласования времени встреча автоматически добавляется в Google Календарь.  

4. **Интеграция с внешними сервисами**  
   - Подключение к CRM-системам для передачи данных о клиентах.  
   - Возможность интеграции с мессенджерами (WhatsApp, Telegram и др.) для расширения каналов коммуникации.  

---

## Используемые сервисы

1. **Qwen**  
   - Применялся для создания системного промпта и формирования базы знаний.  
   - Позволил настроить логику работы ИИ-ассистента и обеспечить качественную генерацию ответов.  

2. **Савви (Suvvy)**  
   - Платформа для разработки мощных AI-ботов с возможностью работы по сложным сценариям.  
   - Обеспечивает надёжность и точность ответов (до 95%).  
   - Интегрируется с CRM, мессенджерами и Google Календарем.  

   Сайт платформы: [https://suvvy.ai/](https://suvvy.ai/)

## Преимущества решения

- **Высокая точность ответов**: Точность ответов ИИ-ассистента достигает 95%, что минимизирует ошибки при общении с клиентами.  
- **Автоматизация рутинных задач**: Запись на встречи и управление календарем выполняются без участия сотрудников.  
- **Повышение удовлетворенности клиентов**: Быстрые и корректные ответы помогают улучшить качество обслуживания.  
- **Масштабируемость**: Решение легко адаптируется под растущие потребности бизнеса.

## Пример использования

1. Клиент обращается через мессенджер с вопросом: "Какие типы окон вы предлагаете?"  
   - ИИ-ассистент анализирует базу знаний и предоставляет подробный ответ о доступных вариантах.  

2. Клиент спрашивает: "Когда можно установить окна?"  
   - Ассистент предлагает свободные временные слоты для установки.  
   - После выбора времени встреча автоматически добавляется в Google Календарь.  

3. Клиент интересуется дополнительными услугами: "Можно ли заказать звукоизоляцию?"  
   - ИИ-ассистент предоставляет информацию о дополнительных опциях и их стоимости.
